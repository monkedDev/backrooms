# 🚪 The Backrooms - Level 0

Интерактивная 3D-веб-версия легендарных **Backrooms** (Уровень 0).

![The Backrooms](https://img.shields.io/badge/Three.js-WebGL-000000?logo=three.js)
![License](https://img.shields.io/badge/license-MIT-blue)

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| **WASD** | Перемещение |
| **Shift** | Бег |
| **Мышь** | Обзор |
| **ESC** | Пауза / меню |

## 🚀 Запуск

### Вариант 1: Локальный сервер (рекомендуется)

```bash
# Через Python
python -m http.server 8000

# Через Node.js (требуется live-server)
npx live-server

# Через PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.

### Вариант 2: GitHub Pages

1. Запушьте репозиторий на GitHub
2. Включите **GitHub Pages** в настройках репозитория
3. Откройте ссылку вида `https://username.github.io/repo-name`

## 🛠 Технологии

- **Three.js** — 3D-графика в браузере
- **PointerLockControls** — FPS-управление
- **Процедурная генерация** — случайная планировка комнат

## 📝 Особенности

- ✅ Аутентичная атмосфера Level 0
- ✅ Жёлтые обои с узором
- ✅ Коричневый ковёр
- ✅ Подвесной потолок с плиткой
- ✅ Случайная генерация лабиринта
- ✅ Динамическое освещение от ламп
- ✅ Туман для атмосферы
